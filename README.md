# Churras do time — Next.js (Tailwind + Framer Motion)

Projeto pronto para rodar localmente.

## Pré-requisitos
- Node.js 18+
- npm

## Instalar e rodar
```bash
npm install
npm run dev
```
Acesse http://localhost:3000

## Build de produção
```bash
npm run build
npm start
```

## Dependências
- next, react, react-dom
- tailwindcss, postcss, autoprefixer
- framer-motion
- lucide-react
